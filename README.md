# Furni – Modern Interior Design Landing Page

A responsive landing page for a modern interior design studio.  
The design and interactions are crafted by us, inspired by a clean, minimalist clothing brand landing page.

## Tech Stack

- **HTML5** – semantic structure for sections like hero, product grid, benefits, and footer.
- **CSS3** – custom layout, typography, and animations (no CSS frameworks).
- **Responsive design** – desktop‑first with breakpoints for tablets and mobile.

## Features

- **Hero section** with strong headline, description, and dual call‑to‑action buttons.
- **“Crafted With Excellent Material”** product row with cards and smooth hover animations.
- **“Why Choose Us”** section explaining key benefits and value.
- **Studio gallery** using overlapping image cards to showcase interiors.
- **Footer** with navigation, support links, and direct GitHub / email contact.
- **Modern hover effects** for buttons and product cards to give a professional, premium feel.

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your browser, or serve the folder with any static server.



## Customization

- Swap the demo images for your own product or interior photos.
- Edit headings, body text, and prices in `index.html` to match your brand.
- Tweak colors and spacing via CSS variables in `styles.css`.
